# Netflix Data Analysis using Python

## Project Overview
This project analyzes Netflix movies and TV shows using Python for Exploratory Data Analysis (EDA) and basic Sentiment Analysis. The project focuses on extracting meaningful insights from Netflix content data through data cleaning, preprocessing, visualization, and NLP techniques.

## Features
- Data Cleaning and Preprocessing
- Handling Missing Values
- Top Directors Analysis
- Top Actors Analysis
- Content Rating Distribution
- Sentiment Analysis using TextBlob
- Interactive Data Visualization using Plotly

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly Express
- TextBlob
- Jupyter Notebook

## Dataset
The dataset contains Netflix titles information including:
- Title
- Director
- Cast
- Release Year
- Rating
- Description
- Genre Information

## Project Workflow
1. Imported and cleaned the Netflix dataset
2. Handled missing values in director and cast columns
3. Split multiple directors and actors for proper analysis
4. Performed grouping and counting operations using Pandas
5. Created pie charts and bar charts using Plotly
6. Applied sentiment analysis on content descriptions using TextBlob
7. Visualized sentiment trends across release years
